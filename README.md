# Psicología + IA - Template para Práctica Profesional

**Template para:** Práctica de psicología con herramientas de IA  
**Estado:** Template configurable  
**Nicho sugerido:** Profesionales Tech, Ingenieros, Developers, Gamers  
**Modelo:** 100% Online/Híbrido - Bootstrapping Total

---

## Sobre Este Template

Este repositorio es un **template completo** para montar una práctica de psicología que integra herramientas de Inteligencia Artificial para transcripción, análisis y documentación clínica.

Incluye: plan de negocio, plantillas clínicas, herramientas de IA, documentos legales (borradores), estrategia de marketing, y guías de implementación.

---

## Inicio Rápido

1. Abrí [`01-INICIO/README.md`](01-INICIO/README.md)
2. Seguí el plan de lanzamiento
3. Usá las plantillas en `04-PLANTILLAS/`

## Sistema de Transcripción

```bash
# Transcribir audios
python -m transcripcion transcribe --input /ruta/a/audios

# Verificar calidad
python -m transcripcion check-quality --input /ruta/a/transcripciones

# Ayuda
python -m transcripcion --help
```

## Modelo de Negocio

### FASE 1: Estudiante

| Aspecto       | Detalle                                         |
| ------------- | ----------------------------------------------- |
| **Servicios** | Coaching de bienestar (NO terapia clínica)      |
| **Target**    | Estudiantes de ingeniería, juniors tech, gamers  |
| **Modalidad** | 100% Online                                     |
| **Legal**     | Divulgar estado estudiante                      |

| Aspecto            | Detalle                                 |

| **Target**         | Profesionales Tech, CTOs                |
| **Diferenciación** | Psicóloga tech-native con IA            |

## Aviso Legal

**FASE 1 (estudiante) - PUEDE:**

- Coaching de bienestar
- Apoyo emocional
- Mindfulness

**NO PUEDE:**

- Diagnósticos clínicos
- Tratamiento de trastornos
- Usar título "psicóloga"

## Cómo Usar Este Template

1. **Fork** este repositorio
2. Reemplaza `[NOMBRE_PROFESIONAL]` con tu nombre en todos los archivos
3. Ajusta precios, horarios y servicios a tu contexto
4. Revisa documentos legales con un abogado de tu jurisdicción
5. Configura las herramientas de IA según `SETUP.md`
