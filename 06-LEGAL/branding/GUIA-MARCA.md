# Guia de Marca

> **Estado:** Por definir
> **Objetivo:** Identidad visual y verbal coherente

## Opciones de Nombre

Basado en el cuestionario, hay 3 direcciones posibles:

### Opcion A: Tech & Minimalista

| Nombre | Disponibilidad | Vibe |
|--------|----------------|------|
| **MindSet Lab** | Verificar | Serio, cientifico |
| **Cortex** | Verificar | Minimalista, tech |
| **Debug Mental** | Verificar | Tech-native, memorable |
| **Null Pointer** (ironico) | Verificar | Insider joke |

**Para quien:** Developers Senior, CTOs, gente que prefiere profesionalismo

---

### Opcion B: Humano & Cercano

| Nombre | Disponibilidad | Vibe |
|--------|----------------|------|
| **Espacio Calma** | Verificar | Calido, protector |
| **Tu Lado B** | Verificar | Personal, introspectivo |
| **Refugio Tech** | Verificar | Safe space |
| **El Reset** | Verificar | Simple, directo |

**Para quien:** Estudiantes, Juniors, gente que busca calidez

---

### Opcion C: Dinamico & Gamified

| Nombre | Disponibilidad | Vibe |
|--------|----------------|------|
| **Level Up Mentality** | Verificar | Coaching, accion |
| **Unlock Potential** | Verificar | Gaming reference |
| **Reset Button** | Verificar | Gaming, reinicio |
| **Mind Patch** | Verificar | Updates, mejora continua |

**Para quien:** Gamers, gente orientada a objetivos

---

### Recomendacion

**Para Fase 1:** Opcion B o C (mas accesible, menos formal)
**Para Fase 2:** Opcion A (mas profesional)

**Top 3 sugeridos:**
1. **Debug Mental** - Tech-native, memorable, diferenciado
2. **Reset Button** - Simple, gaming-friendly, action-oriented
3. **Espacio Calma** - Si queres algo mas tradicional

## Paleta de Colores

### Opcion 1: Dark Mode (Tech-native)

```
Primario:     #1E1E1E (Negro suave - fondo)
Secundario:   #00FF00 (Verde terminal - acentos)
Terciario:    #4EC9B0 (Cyan - CTAs)
Texto:        #D4D4D4 (Gris claro)
Alerta:       #FF5555 (Rojo suave)
```

**Uso:** Apela directamente a developers, refuerza identidad tech

---

### Opcion 2: Minimalista Calm

```
Primario:     #FFFFFF (Blanco - fondo)
Secundario:   #2D3748 (Gris oscuro - texto)
Terciario:    #38B2AC (Teal - acentos)
Cuaternario:  #EDF2F7 (Gris muy claro - fondos secundarios)
```

**Uso:** Mas tradicional, transmite calma y profesionalismo

---

### Opcion 3: Warm Tech

```
Primario:     #1A202C (Azul muy oscuro - fondo)
Secundario:   #F6AD55 (Naranja calido - acentos)
Terciario:    #68D391 (Verde menta - exito)
Texto:        #E2E8F0 (Gris claro)
```

**Uso:** Balance entre tech y calidez humana

---

### Recomendacion

**Para Fase 1:** Opcion 1 (Dark Mode) si el target es developers
**Alternativa:** Opcion 3 (Warm Tech) para balance

## Tipografia

### Para Digital

**Headlines:**
- Inter (Google Fonts - gratis)
- SF Pro Display (si es Apple ecosystem)

**Body:**
- Inter (mismo que headlines para consistencia)
- Source Sans Pro (alternativa)

**Codigo/Monospace:**
- Fira Code
- JetBrains Mono

### Para Redes Sociales

Usar las fuentes de Canva que mejor imiten:
- Headlines: Algo bold, sans-serif
- Body: Legible, no muy pequeno

## Tono de Voz

### Caracteristicas

| Si | No |
|----|----|
| Directo | Ambiguo |
| Casual pero profesional | Demasiado formal |
| Tech-friendly | Jerga excesiva de psicologia |
| Empatico sin ser meloso | Frio o distante |
| Honesto | Exagerado |

### Ejemplos

**En lugar de:**
> "Ofrecemos servicios terapeuticos integrales para el manejo del estres laboral"

**Decir:**
> "Te ayudo a no quemarte en el trabajo"

---

**En lugar de:**
> "Agenda tu consulta inicial"

**Decir:**
> "Escribime cuando quieras"

---

**En lugar de:**
> "Especialista en salud mental ocupacional"

**Decir:**
> "Entiendo tu mundo porque vivi rodeada de devs"

## Elementos Visuales

### Iconografia

Usar iconos de:
- Feather Icons (minimalista)
- Heroicons (moderno)
- Phosphor Icons (versatil)

**Estilo:** Linea fina, minimalista, no muy decorativo

### Imagenes

**Si usar:**
- Setups de trabajo (home office)
- Pantallas con codigo (desenfocado)
- Espacios minimalistas
- Naturaleza (para balance)

**No usar:**
- Stock photos genericas de "persona sonriendo"
- Divanes de psicologo
- Imagenes muy corporativas

### Mockups

Para presentar el servicio, considerar:
- Screenshots de WhatsApp (mockup)
- Laptop con videollamada
- Calendario/agenda

## Checklist de Consistencia

Antes de publicar cualquier cosa, verificar:

- [ ] ¿Usa los colores de la paleta?
- [ ] ¿El tono es consistente con la marca?
- [ ] ¿Se ve bien en movil?
- [ ] ¿Tiene call-to-action claro?
- [ ] ¿Representa bien al target (tech)?
