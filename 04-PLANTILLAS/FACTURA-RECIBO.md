# Recibo de Pago

> **Uso:** Entregar despues de recibir pago
> **Formato:** Copiar y completar, enviar por WhatsApp o email

---

```

Recibo #: [NUMERO]

Recibi de: [NOMBRE DEL CLIENTE]
La suma de: Gs. [MONTO]
En concepto de: [DESCRIPCION]

Forma de pago: [Transferencia/Efectivo/Otro]

________________________
[TU NOMBRE]
CI: [TU CI]
```

## Sistema de Numeracion

**Formato sugerido:** R-YYYY-###

Donde:
- R = Recibo
- YYYY = Ano
- ### = Numero secuencial

**Ejemplos:**
- R-2026-001 (primer recibo del 2026)
- R-2026-002 (segundo recibo)
- R-2026-015 (recibo numero 15)

## Notas Importantes

1. **Guardar copia** de todos los recibos emitidos
2. **Registrar ingresos** para declaracion de impuestos
3. **No es factura legal** - si el cliente necesita factura formal, consultar con contador
4. **RUC:** Si creces, considerar obtener RUC y emitir facturas formales
